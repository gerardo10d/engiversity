Bienvenido a la aplicación web Engiversity

Este proyecto de desarrollo tecnológico está siendo elaborado por los profesores:

  • Gerardo Andrés Dorado Jurado, en calidad de investigador principal (ver hoja de vida).
  • José Daniel Muñoz Martínez, en calidad de coinvestigador (ver hoja de vida).
  • Juan Diego Muñoz Ordoñez, en calidad de coinvestigador (ver hoja de vida).

El objetivo es crear una aplicación web para facilitar los procedimientos de cálculo y diseño utilizados en los componentes de la ingeniería civil, como son los pavimentos, suelos, hidráulica, tránsito, entre otros. La aplicación se desarrolla utilizando HTML, CSS y JavaScript con una interfaz gráfica de fácil uso y visualización de resultados. También se tiene el propósito de proporcionar a estudiantes e ingenieros consultores una herramienta para analizar procedimientos y resultados recurrentes en la disciplina. Se pretente indagar sobre los procedimientos y métodos de diseño para seleccionar los más representativos de cada componente de la ingeniería civil. Estos se integran en una aplicación web didáctica para apoyar tanto el trabajo del profesional como la enseñanza de la ingeniería. El proyecto busca satisfacer una necesidad en la industria y la academia de ingeniería civil.
