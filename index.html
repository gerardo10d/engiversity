Bienvenido a la aplicación web Engiversity

Este proyecto de desarrollo tecnológico del profesor Gerardo Andrés Dorado Jurado (ver hoja de vida) propone la creación de una aplicación web para facilitar los procedimientos de cálculo y diseño utilizados en las áreas de la ingeniería civil, como son los pavimentos, estructuras, geotecnia e hidráulica. La aplicación se desarrolla utilizando HTML, CSS y JavaScript con una interfaz gráfica de fácil uso y visualización de resultados. El objetivo es proporcionar a estudiantes e ingenieros consultores una herramienta para analizar procedimientos y resultados recurrentes en la disciplina. Se pretente indagar sobre los procedimientos y métodos de diseño para seleccionar los más representativos de cada área de la ingeniería civil. Estos se integran en una aplicación web didáctica para apoyar tanto el trabajo del profesional como la enseñanza de la ingeniería. El proyecto busca satisfacer una necesidad en la industria y la academia de ingeniería civil.
