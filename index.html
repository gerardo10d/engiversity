Bienvenido a la aplicación web Engiversity

Esta aplicación web está siendo desarrollada por el Ing. Gerardo Andrés Dorado Jurado (ver hoja de vida).

El objetivo es facilitar los procedimientos de cálculo y diseño utilizados en las áreas de la ingeniería civil, como son geotecnia, pavimentos, hidráulica, tránsito, entre otros. La aplicación se desarrolla utilizando HTML, CSS y JavaScript con una interfaz gráfica de fácil uso y visualización de resultados. También se tiene el propósito de proporcionar a estudiantes e ingenieros consultores una herramienta para analizar procedimientos y resultados recurrentes en la disciplina. Se pretente indagar sobre los procedimientos y métodos de diseño para seleccionar los más representativos de cada componente de la ingeniería civil. Estos se integran en una aplicación web didáctica para apoyar tanto el trabajo del profesional como la enseñanza de la ingeniería. El proyecto busca satisfacer una necesidad en la industria y la academia de ingeniería civil.
